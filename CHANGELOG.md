# 변경 기록 (Changelog)

## [0.1.1] - 2024-07-26

### 추가
- 메인 페이지 '제공 서비스' 섹션에 '특수 철거' 항목 추가.

- 5가지 기본 서비스 항목 추가 (상가, 사무실, 가벽, 식당, 지원금 컨설팅)
